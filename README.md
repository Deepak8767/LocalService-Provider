LocalService-Provider is a **Java-based web application** that connects users with nearby local service providers such as plumbers, electricians, cleaners, and helpers.  
The platform allows users to book services online and enables service providers to view and manage bookings through their dashboard.

This project is developed as a **Dynamic Web Project using JSP, Servlets, JDBC, and MySQL**.
